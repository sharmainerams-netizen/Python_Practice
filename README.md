# Python-Projects-Just4Fun
1. Ceasar Cipher : This a program I learnt to develop from a python course (FreeCodeCamp) , i added user input and very basic structure. Essentially its a program that encodes/decodes text by altering the alphabets with a shift e.g with a shift of 3 - Encoding: 'a' in your text will change to 'd' ; Decoding: 'e' will be 'b' and so on.It's a very simple encrytion technique that dates back to around the 1st century. Very useful for basic information security.
